Materi HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan menyusun halaman web. Berikut adalah beberapa elemen dasar dalam HTML:

Contoh Elemen HTML Dasar

1. Heading

<h1>Ini adalah heading level 1</h1>
<h2>Ini adalah heading level 2</h2>
<h3>Ini adalah heading level 3</h3>

2. Paragraf

<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>

3. Tautan (Link)

<a href="https://www.example.com">Kunjungi Example</a>

4. Gambar

<img src="img/example.jpg" alt="Contoh Gambar" width="500" />

5. Daftar

Daftar Tidak Berurut (Unordered List)

<ul>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>

Daftar Berurut (Ordered List)

<ol>
   <li>Item 1</li>
  <li>Item 2</li>
   <li>Item 3</li>
</ol>

6. Tabel

<table border="1">
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Budi</td>
    <td>30</td>
  </tr>
</table>

7. Formulir

<form action="/submit" method="post">
  <label for="name">Nama:</label>   <input type="text" id="name" name="name"><br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br>
  <input type="submit" value="Submit">
</form>

8. Video

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
</video>

9. Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
