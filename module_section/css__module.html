Materi CSS

CSS (Cascading StyleSheet) adalah bahasa standar untuk Memberikan gaya atau styling pada halaman web. Berikut adalah konsep konsep yang ada di CSS :

Contoh Properti CSS Dasar

1. Warna Teks

color: red;
background-color: yellow;

2. Ukuran Teks

font-size: 16px;
font-family: Arial, sans-serif;

3. Margin dan Padding

margin: 10px;
padding: 20px;

4. Border

border: 1px solid black;
border-radius: 5px;

5. Posisi

position: relative;
top: 20px;

6. Display

display: block;
display: inline-block;

7. Animasi

transition: all 0.3s ease-in-out;
animation: slide 1s infinite;

8. Gradien

background: linear-gradient(to right, red, yellow);

9. Transformasi

transform: rotate(45deg);
transform: scale(1.2);
